   ## Overview                                                                            
   A skill for "distilling" AI agent memory.                                              
   Discarding the old idea of "deleting unnecessary info," it specializes in **extracting 
 valuable insights from noise**.                                                          
                                                                                          
   ## Installation                                                                        
   1. Install .skill from ClawHub or copy to workspace/skills/public/.                    
   2. `read skills/public/agents-refresh-md-en-v1.1.1/SKILL.md` to trigger.               
   3. Cron auto from manifest.yaml or manual `cron add` (copy from Quick Start).          
                                                                                          
   ## Usage                                                                               
   - **Auto:** 6AM cron reloads AGENTS/IDENTITY/SOUL + MEMORY distillation.               
   - **Manual:** `session_status` ‚Üí edit HEARTBEAT.md ‚Üí cron test run.                    
   - **Heartbeat:** 4h rotate (9AM/1PM/8PM) for long-session personality keep.
