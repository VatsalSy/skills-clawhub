# 文件处理技能

自动识别和处理用户发送的文件。

## 支持格式

- PDF (pdf)
- Excel (xlsx, xls)
- CSV (csv)
- Word (docx)
- 图片 (jpg, png, jpeg)
- 文本 (txt)

## 功能

### 1. PDF 读取
提取文字、表格、页码信息

### 2. Excel/CSV 处理
读取数据、统计、筛选

### 3. OCR 文字识别
图片转文字

### 4. 文档摘要
长文本自动摘要

## 依赖

```bash
pip install pdfplumber openpyxl python-docx pytesseract pillow
```

## 使用方法

直接发送文件给小乡，根据格式自动处理。
