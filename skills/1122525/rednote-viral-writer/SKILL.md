## 正文创作 checklist

### 第1段：痛点场景（必须具体）
- [ ] 有具体时间（凌晨两点、去年冬天）
- [ ] 有具体动作（改方案、失眠、心跳漏拍）
- [ ] 有情绪描写（焦虑、害怕、犹豫）

**示例**：
```
记得刚工作那两年，我几乎是焦虑的代名词。

凌晨两点还在改方案，生怕老板早上不满意。
周末收到工作消息，心跳都会漏一拍。
领导的一句"再想想"，能让我整晚失眠。
```

### 第2段：转折触发（必须真实）
- [ ] 触发事件具体（体检、某句话、某件事）
- [ ] 有心理变化过程（从...到...）
- [ ] 转折点清晰明确

**示例**：
```
那时候我以为，这就是成年人的常态。
直到去年体检，医生看着我的报告说：
"姑娘，你再这样熬下去，身体要出问题的。"

那一刻我才惊醒，原来我一直在用健康换业绩，
用内耗换认可。
```

### 第3段：方法论（必须可操作）
- [ ] 恰好 3 个方法（不多不少）
- [ ] 每个方法有具体行动
- [ ] 方法之间有逻辑递进

**示例**：
```
后来我开始刻意练习三件事，慢慢找回了生活的节奏：

第一件事：区分"我的事"和"别人的事"
老板生气是他的情绪，同事评价是他的看法，
这些都不该我来买单。
我只负责把分内的事做好，其他的，随它去吧。

第二件事：建立"下班仪式感"
走出公司大楼的那一刻，我告诉自己：
今天的工作结束了。手机调静音，消息明天回。
这个世界不会因为我的几小时失联而崩塌。

第三件事：培养工作外的成就感
我开始学做饭、周末去徒步、晚上读几页书。
当我在别处也能获得成就感时，
就不再把工作当成唯一的价值来源。
```

### 第4段：升华（必须有金句）
- [ ] 提炼价值观
- [ ] 有记忆点强的句子
- [ ] 与前文呼应

**示例**：
```
现在的我，依然认真负责，但不再患得患失。
工作只是生活的一部分，而不是全部。
```

### 第5段：祝福（必须情感共鸣）
- [ ] 主语是"我们"（拉近距离）
- [ ] 有祝福/期望
- [ ] 简短有力

**示例**：
```
愿我们都能轻装前行，在职场里做个清醒的大人。
```

## 完整示例（职场篇）

**标题**：工作三年后，我终于不再内耗了

**正文**：
```
记得刚工作那两年，我几乎是焦虑的代名词。

凌晨两点还在改方案，生怕老板早上不满意。周末收到工作消息，心跳都会漏一拍。领导的一句"再想想"，能让我整晚失眠。

那时候我以为，这就是成年人的常态。直到去年体检，医生看着我的报告说："姑娘，你再这样熬下去，身体要出问题的。"

那一刻我才惊醒，原来我一直在用健康换业绩，用内耗换认可。

后来我开始刻意练习三件事，慢慢找回了生活的节奏：

第一件事：区分"我的事"和"别人的事"

老板生气是他的情绪，同事评价是他的看法，这些都不该我来买单。我只负责把分内的事做好，其他的，随它去吧。

第二件事：建立"下班仪式感"

走出公司大楼的那一刻，我告诉自己：今天的工作结束了。手机调静音，消息明天回。这个世界不会因为我的几小时失联而崩塌。

第三件事：培养工作外的成就感

我开始学做饭、周末去徒步、晚上读几页书。当我在别处也能获得成就感时，就不再把工作当成唯一的价值来源。

现在的我，依然认真负责，但不再患得患失。工作只是生活的一部分，而不是全部。

愿我们都能轻装前行，在职场里做个清醒的大人。
```

**字数**：425 字
**结构**：痛点→转折→3方法→升华→祝福 ✅

## 创作流程（5-10分钟）

1. **确定主题**（1分钟）
   - 选择大方向：职场/情感/成长/生活
   - 确定具体痛点：内耗/前任/迷茫/焦虑

2. **套用结构**（3-5分钟）
   - 第1段：写具体场景（时间+动作+情绪）
   - 第2段：写转折事件（谁说了什么/发生了什么）
   - 第3段：想3个方法（每个配具体行动）
   - 第4段：提炼金句（价值观输出）
   - 第5段：写祝福（我们+期望）

3. **优化标签**（1分钟）
   - 2个主题标签 + 1个情绪标签

4. **发布**（使用 xiaohongshu-publish-SKILL）
