# OpenClaw Browser Toggle Skill

> 一键启用/禁用 OpenClaw 内置浏览器，无需手动修改配置文件

**Name:** browser-toggle  
**Version:** 1.0.0  
**Author:** AI Assistant  
**License:** MIT  
**Description:** 一键启用/禁用 OpenClaw 内置浏览器

---

## Installation

```bash
bash setup.sh
```

## Usage

```bash
openclaw-browser --enable
openclaw-browser --disable
openclaw-browser --status
```

## Features

- ✅ 一键启用/禁用内置浏览器
- ✅ 自动备份配置文件
- ✅ 失败自动恢复
- ✅ 支持可视化/无头模式
- ✅ 跨平台支持

## Requirements

- Python 3.8+
- OpenClaw 2026.2.26+
- Chrome/Chromium
