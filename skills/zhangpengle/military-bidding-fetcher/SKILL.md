# 军工采招商机猎手

---

## 🛠️ 核心功能

### 1. 多源同步监测
集成三大权威军标平台，实现全方位覆盖：
* **全军武器装备采购信息网 (WEAIN)**: 聚焦武器装备研发、维修、配套。
* **军队采购网 (PLAP)**: 覆盖通用物资、工程建设及保障服务。
* **国防科技大学采购网 (NUDT)**: 侧重科研实验、信息化系统及软件开发。

### 2. 智能推荐引擎 (Recommendation Logic)
系统根据项目名称中的核心关键词，自动标注推荐等级：
* 🔴 **高 (High)**: 包含 `意向公开`、`模型`、`仿真`、`AI`、`智能化`、`决策分析`。
* 🟡 **中 (Medium)**: 包含 `系统`、`信息化`、`集成`、`软件定制`。
* ⚪ **普通**: 其他符合核心关键词匹配的项目。

### 3. 数据精准清洗
* **状态过滤**: 自动屏蔽 `中标公告`、`结果公示`、`废标公告`，确保用户只关注可投标项目。
* **内容降噪**: `_clean_description` 逻辑自动剔除 HTML/CSS 标签、JS 乱码，提取 20-50 字的核心业务简述。
* **负向拦截**: 自动过滤服装、食品、医疗、装修等低相关度干扰项。

---

## 📖 接口与参数说明 (Manifest)

### `fetch_all_bidding`
**功能**: 一键执行全平台汇总抓取并生成 Excel。

| 参数名 | 类型 | 必填 | 默认值 | 描述 |
| :--- | :--- | :--- | :--- | :--- |
| `keywords` | Array | 否 | 系统预设 | 核心关键词列表 (如: `["雷达", "数字孪生"]`) |
| `exclude_keywords` | Array | 否 | 系统预设 | 需要排除的行业词 (如: `["物业", "食材"]`) |
| `date` | String | 否 | 当天日期 | 检索的目标日期 (格式: `YYYY-MM-DD`) |
| `auto_latest` | Boolean | 否 | `true` | 是否自动探测各平台最新的发布日期 |

---

## 📂 输出规范

执行完成后，系统将在工作区生成一份 **`商机信息汇总_YYYY-MM-DD.xlsx`**，包含以下 Sheet：

* **Sheet 1: 全军武器装备采购信息网**
    * 包含：涉密等级、截止日期、详情链接、匹配关键词、推荐等级。
* **Sheet 2: 军队采购网**
    * 包含：地区、采购方式、清洗后的项目描述、推荐等级、备注。
* **Sheet 3: 国防科大采购信息网**
    * 包含：公告类型、详情链接、发布日期、分析备注。
