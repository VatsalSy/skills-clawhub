# 爽文/漫剧风格指南 (Manga Drama Guide)

## 适用场景
- 都市逆袭、霸总、打脸、赘婿等"爽文"类型
- 有声漫画/漫剧形式的短剧
- 要求"强情绪、高反转、快节奏"的项目

## 核心特征

### 爽文三要素
1. **强情绪**：每个镜头都要有情绪标签，旁白带感情色彩
2. **高反转**：每1-2集至少一次反转，"以为他弱其实他强"
3. **快节奏**：删除所有不必要的过渡，直奔冲突和爽点

### 旁白风格
爽文/漫剧的旁白不是客观叙述，而是带情绪引导的解说：

| 情绪 | 旁白风格 | 示例 |
|------|---------|------|
| 铺垫压抑 | 沉重、低沉 | "三年了...他在这张病床上，已经躺了整整三年" |
| 打脸前奏 | 不屑、嘲讽转震惊 | "所有人都在嘲笑他...直到——" |
| 爆发时刻 | 激昂、热血 | "这一刻！全场鸦雀无声！" |
| 反转揭秘 | 神秘、震撼 | "没人知道...这个被所有人看不起的废物，竟然是——" |

### 情绪标签系统

每个镜头标注情绪强度（1-5级）：

| 级别 | 情绪类型 | 用法 |
|------|---------|------|
| ⭐ | 平静/日常 | 过渡场景 |
| ⭐⭐ | 好奇/疑惑 | 铺垫 |
| ⭐⭐⭐ | 紧张/期待 | 蓄力 |
| ⭐⭐⭐⭐ | 震惊/愤怒/激动 | 爆发 |
| ⭐⭐⭐⭐⭐ | 狂喜/绝望/极致震撼 | 高潮 |

### 节奏模式
```
压抑(2-3镜头) → 蓄力(1-2镜头) → 爆发(1镜头) → 余韵(1镜头) → 新钩子
```

## 漫剧专用分镜技法

### 速度线/冲击感
- 动作镜头加"动态模糊感"、"冲击波纹"
- 配合"破空呼啸声"等音效

### 分屏/对比
- 两个角色同时反应：`画面左侧A震惊后退，画面右侧B冷笑前进`

### 定格+旁白
- 关键时刻画面定格，配旁白解说
- `（定格）旁白：这一拳...蕴含了三年的屈辱`

### 黑屏+文字
- 集尾反转用黑屏+大字
- `黑屏，中央白色大字：「三天后，世界杯决赛」`

## AI绘图Prompt增强

爽文场景的提示词需要额外增强词：

| 场景类型 | 增强词 |
|---------|--------|
| 打脸/碾压 | `高对比度光影，戏剧性打光，俯角/仰角强化` |
| 热血/爆发 | `动态模糊，汗水飞溅，速度线，冲击波` |
| 震惊/反转 | `瞳孔放大特写，背景虚化，聚焦面部表情` |
| 压抑/屈辱 | `低饱和度，阴影覆盖，冷色调，低角度` |
| 觉醒/蜕变 | `金色/白色光芒，逆光，头发飘扬，气场扩散` |
