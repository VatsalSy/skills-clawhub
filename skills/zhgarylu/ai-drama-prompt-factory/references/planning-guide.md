# 策划指南 (Planning Guide)

## 小说改编路径

### 输入处理
- 短篇（<3万字）：直接通读
- 中篇（3-10万字）：分章节读取
- 长篇（>10万字）：建立文档索引（参见 doc-reader-guide.md），分批处理

### 故事大纲输出格式

```markdown
一、故事大纲

**1. 基础信息**
作品名称：**{名称}**
写作视角：{第一人称/第三人称全知/等}

**2. 时空背景**
{年代、地点、社会环境、特殊设定，150-300字}

**3. 叙事结构**
{线性/非线性，分集数量（每集1-3分钟），叙事递进模式，传播逻辑}

**4. 故事核心**
**核心冲突**：{主角+困境+对抗力量+双重压力+逆袭路径，含视觉化元素提示}

**主要角色**：
1. **{角色名}**（性别）：身份为{身份}，{表面特征}，{内在特质}，{核心能力}
2. ...

**次要角色**：
- {角色名}（性别）：{身份}，{特征}，{故事功能}
- ...

**主要道具**：
**{道具名}**
- 外观：{详细视觉描述}
- 功能：{剧情功能}
- 首次出现：第{N}集

**5. 分集规划**
第1集「{标题}」：{核心事件}。钩子：{悬念/反转}
第2集「{标题}」：{核心事件}。钩子：{悬念/反转}
...
```

### 分集规划原则
- 每集时长1-3分钟（15-45个镜头）
- 前3秒必须有冲突或悬念（防划走）
- 每集结尾必须有强钩子（cliffhanger）
- 节奏：起承转合不超过4集一个小高潮
- 大高潮放在全剧2/3处

## 原创路径

### 主题分析输出
```
核心卖点：{一句话Logline}
目标受众：{画像描述}
情绪曲线：{起伏节奏}
平台适配：{抖音/快手/TikTok建议}
变现潜力：{评估}
```

### 故事结构设计
- 三幕结构（短剧常用）：建置→对抗→解决
- 五幕结构（多集长篇）：铺垫→升级→危机→高潮→余韵
- 核心冲突必须在第1集前30秒内建立

### 爽文/短剧类型公式

| 类型 | 核心公式 | 节奏 |
|------|---------|------|
| 都市逆袭 | 底层→觉醒→碾压→巅峰 | 每1-2集一次打脸 |
| 霸总甜宠 | 偶遇→误会→心动→在一起 | 高甜高虐交替 |
| 穿越重生 | 重生→复仇→逆袭→圆满 | 知晓未来的信息差制造爽感 |
| 赘婿逆袭 | 受辱→隐忍→爆发→碾压 | 前期压抑越深后期爆发越爽 |
