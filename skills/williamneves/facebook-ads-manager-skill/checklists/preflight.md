# Preflight Checklist

Execute antes de qualquer acao de escrita.

- [ ] Objetivo da acao esta claro e mensuravel.
- [ ] `act_id` esta no formato `act_<id>`.
- [ ] Baseline coletado com insights relevantes.
- [ ] Tool e parametros foram revisados com nome exato.
- [ ] Mudanca minima foi definida (sem salto agressivo).
- [ ] Plano de rollback esta registrado.
- [ ] Impacto esperado foi comunicado ao usuario.
