# Postflight Checklist

Execute imediatamente apos qualquer escrita.

- [ ] IDs alterados foram confirmados e registrados.
- [ ] Estado final foi verificado por leitura (`get_*_by_id`).
- [ ] Resultado esperado e risco residual foram comunicados.
- [ ] Janela de monitoramento foi definida.
- [ ] Condicao de rollback foi definida com gatilho claro.
- [ ] Proximo checkpoint com usuario foi combinado.
