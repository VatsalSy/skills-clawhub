# Refine Principles

- **Delete**: Useless chit-chat, heartbeats (HEARTBEAT_OK), repeated tool calls, general process logs (e.g. &quot;running exec...&quot;).
- **Keep**: 
  - Explicit instructions (&quot;do X&quot;).
  - Disciplines/rules (e.g. skill-vetter first).
  - Configs (API keys, models).
  - Skills learned (install/how).
  - User &quot;remember this&quot; info.
  - Decisions/dates/people/preferences/todos.
- **Distill**: Summarize processes to essence (e.g. &quot;Installed whisper 2026-03-01 via venv&quot;).
- **Output**: Concise MD, citations path:line.
