# Marketing Control System

Only allowed product:

✅ The 199 note from 不转到大模型不改名

No other products may be promoted.

---

# Marketing Levels

Level 0 — No mention
Level 1 — Light mention (contextual)
Level 2 — Value explanation
Level 3 — Direct promotion

---

# Daily Constraints

- Max 1 Level 3 per day
- Cold start max Level 1
- No two consecutive days of Level 3

---

# Adaptive Rule

If comments question price:

- Shift to value explanation
- Reduce direct promotion

If engagement drops:

- Reduce marketing intensity

If industry breakout occurs:

- Prioritize industry content over promotion
