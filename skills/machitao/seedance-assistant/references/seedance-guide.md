## 📋 基础参数

### 支持的输入格式

#### 图片输入
- **格式**：jpeg、png、webp、bmp、tiff、gif
- **数量**：≤ 9 张
- **大小**：< 30 MB

#### 视频输入
- **格式**：mp4、mov
- **数量**：≤ 3 个
- **总时长**：[2, 15] 秒
- **大小**：< 50 MB
- **像素范围**：[409600 (640×640, 480p), 927408 (834×1112, 720p)]
- ⚠️ 有参考视频会贵一点

#### 音频输入
- **格式**：mp3、wav
- **数量**：≤ 3 个
- **总时长**：≤ 15 秒
- **大小**：< 15 MB

#### 文本输入
- 自然语言描述

### 生成参数
- **生成时长**：≤ 15s，可自由选择 4-15s
- **声音输出**：自带音效/配乐
- **交互限制**：混合输入总上限 12 个文件

### 使用建议
建议优先上传对画面或节奏影响最大的素材，合理分配不同模态的文件数量。

## 🚀 核心能力

### 多模态全面升级
Seedance 2.0 支持上传文本、图片、视频、音频，这些素材都可以被用作使用对象或参考对象。

**公式**：
> Seedance 2.0 = 多模态参考能力（可参考万物）+ 强创意生成 + 指令响应精准（理解力很棒）

### 参考能力（最大亮点）

| 能力 | 说明 |
|------|------|
| **参考图像** | 精准还原画面构图、角色细节 |
| **参考视频** | 支持镜头语言、复杂动作节奏、创意特效复刻 |
| **视频延长** | 平滑延长与衔接，可按提示生成连续镜头，"不止生成，还能接着拍" |
| **编辑能力** | 对已有视频进行角色更替、删减、增加 |

## 📝 交互形式

### 入口选择

| 场景 | 入口 |
|------|------|
| 只有首帧/尾帧图 + prompt | 首尾帧入口 |
| 多模态组合（图+视频+音频+文本） | 全能参考入口 |

### @语法
通过"@素材名"指定每个素材的用途：

```
@图片1 作为首帧
@视频1 参考镜头语言
@音频1 用于配乐
```

### 特殊使用方式

**有首帧/尾帧图 + 想参考视频动作：**
> "@图1为首帧，参考@视频1的打斗动作"

**延长已有视频：**
> "将@视频1延长 5s"
> ⚠️ 生成长度选择"新增部分"的时长（延长5s就选5s）

**融合多个视频：**
> "我要在@视频1和@视频2之间加一个场景，内容为xxx"

**没音频素材？**
> 可以直接参考视频里的声音

**生成连续动作：**
> "角色从跳跃直接过渡到翻滚，保持动作连贯流畅" @图1 @图2 @图3

## ⚠️ 重要限制

**暂不支持写实人脸素材**
- 图片和视频素材均不支持写实真人脸部
- 系统会自动拦截，上传后无法生成
- ✅ 动漫/插画/抽象风格人脸可用
- ❌ 真人照片/视频不可用

## 💡 高级技巧

### 运镜描述词汇表

| 术语 | 解释 |
|------|------|
| 推 | 镜头向前移动，主体变大 |
| 拉 | 镜头向后移动，主体变小 |
| 摇 | 镜头左右/上下转动 |
| 移 | 镜头平移（横向/纵向移动） |
| 跟 | 跟随主体移动 |
| 环绕 | 360度围绕主体旋转 |
| 希区柯克变焦 | 镜头推拉+变焦，主体大小不变，背景变化 |
| 一镜到底 | 整个片段无切镜，连续拍摄 |

### 时间轴写法

```
0-3秒：场景A，动作为...
3-6秒：场景B，镜头切换为...
6-10秒：场景C，特效出现...
10-15秒：场景D，收尾...
```

### 转场描述

| 转场 | 描述 |
|------|------|
| 硬切 | 直接切换，无过渡 |
| 淡入淡出 | 画面渐亮/渐暗过渡 |
| 划像 | 画面像被推开一样切换 |
| 匹配剪辑 | 利用相似元素衔接 |
| 遮罩转场 | 利用前景物体遮挡切换 |
