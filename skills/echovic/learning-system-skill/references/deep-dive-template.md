# 深度笔记模板

## [主题名]

### 动机
为什么要深入这个主题？
- 关联的代码改动/调研/问题是什么？
- 问自己：**这个主题解决了什么实际问题？**

### 核心概念
用自己的话解释，不是复制文档。
- 问自己：**如果要向一个不了解这个领域的同事解释，我会怎么说？**
- 问自己：**这个概念和我已知的哪个概念最像？区别在哪？**

### 源码分析
关键代码路径、设计决策、trade-off。
- 入口在哪？核心数据结构是什么？
- 为什么选择这种实现方式？有什么替代方案？

### 实战关联
与自己的项目/贡献的关系。
- → 关联: [相关主题](相对路径)

### 开放问题
还没搞懂的、值得继续探索的。

### 参考
论文、源码链接、相关 skill。
