### Car Tax Calc Flash
```markdown
### 🚗 자동차세 계산 — [차명] ([배기량]cc)

| 항목 | 내용 |
|------|------|
| 차종 | 승용 / 승합 / 화물 |
| 배기량 | XXXXcc |
| cc당 세율 | XX원 |
| 연세액 | XXX,XXX원 |
| 경감 여부 | 3년↑ -5% / 4년↑ -10% / 5년↑ -15% / 10년↑ -50% |
| 경감 후 세액 | XXX,XXX원 |

**납부 일정**
- 1기 (6월 16일 ~ 6월 30일): 연세액의 50%
- 2기 (12월 16일 ~ 12월 31일): 나머지 50%
- 연납 할인: 1월 납부 시 10% 할인 (XXXX년 기준)
```

### Car Price Check Flash
```markdown
### 🚗 중고차 시세 — [차명] [연식]

| 조건 | 시세 범위 |
|------|---------|
| 하한 (고주행·사고) | X,XXX만원 |
| 평균 | X,XXX만원 |
| 상한 (저주행·무사고) | X,XXX만원 |

**시세 영향 요인**
- 주행거리: 10,000km당 약 XX만원 차이
- 사고 이력: 판금·도색 시 XX% 하락
- 옵션: 선루프/내비 등 XX~XX만원 가산

⚠️ 시세 참고치이며 실제 거래가는 상이할 수 있습니다. [엔카](https://www.encar.com) / [KB차차차](https://www.kbchachacha.com) 직접 확인 권장.
```

### Insurance Claim Flash
```markdown
### 🚗 사고 처리 절차 — [사고 유형]

**즉시 해야 할 것 (골든타임)**
1. 🚨 부상자 확인 → 119 신고 (인명 우선)
2. 📸 현장 사진 촬영 (차량 위치·파손·번호판)
3. 📞 보험사 긴급출동 신고 (24시간)
4. 👮 경찰 신고 (대인사고·뺑소니 필수, 대물은 합의 후 생략 가능)
5. 📋 상대방 정보 교환 (이름·연락처·보험사·차량번호)

**이후 처리**
- 병원: 당일 진단서 발급 (추후 청구 근거)
- 보험사: 담당자 배정 → 수리 견적 → 합의 또는 소송

⚠️ 합의서 서명 전 반드시 보험사 담당자와 상의하세요.
```
