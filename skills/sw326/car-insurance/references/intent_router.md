# 인텐트 라우터 (10개 모드)

> 공통 분류 규칙: [`../../common/intent_router_base.md`](../../common/intent_router_base.md)

## 분류 규칙 (공통)
사용자 요청을 아래 10개 인텐트 중 하나로 분류한다.
복합 요청은 주 인텐트 + 보조 인텐트로 처리한다.
파라미터 부족 시 1회만 질문. 추측 가능하면 추측 후 명시.

---

## 인텐트 목록

### 1. Car Tax Calc (자동차세 계산)
- **표현:** "자동차세 얼마야?", "차 세금 계산", "연세액 알려줘", "내 차 세금"
- **산출물:** 배기량/차종 기반 자동차세 즉시 계산 + 납부 일정
- **최소 데이터:** 차종(승용/승합/화물), 배기량(cc) 또는 정원
- **깊이:** Flash only
- **특이사항:** 배기량 모르면 차명으로 추정 후 명시

### 2. Car Price Check (중고차 시세)
- **표현:** "중고차 시세", "이 차 중고로 얼마야?", "시세 조회해줘", "중고 가격"
- **산출물:** 차종·연식별 중고차 시세 범위 Flash
- **최소 데이터:** 차종, 연식
- **깊이:** Flash only

### 3. Insurance Quote (보험료 조회)
- **표현:** "자동차보험 얼마야?", "보험료 견적", "갱신하면 얼마?", "다이렉트 보험료"
- **산출물:** 조건별 보험료 참고 범위 + 주요 영향 요인
- **최소 데이터:** 차종, 운전자 나이
- **깊이:** Flash only

### 4. Mandatory Insurance (의무보험)
- **표현:** "책임보험 꼭 가입해야 해?", "의무보험 뭐야?", "미가입 걸리면?", "대인배상"
- **산출물:** 의무보험 종류 + 한도 + 미가입 패널티
- **최소 데이터:** 없음
- **깊이:** Flash only

### 5. Insurance Claim (사고 처리)
- **표현:** "사고 났어", "사고 처리 어떻게?", "보험 청구 방법", "접촉사고 합의"
- **산출물:** 사고 유형별 처리 절차 단계 Flash
- **최소 데이터:** 사고 유형(단독/대물/대인/뺑소니 등)
- **깊이:** Flash + Deep-Dive 요청 시

### 6. Maintenance Cost (유지비)
- **표현:** "유지비 얼마야?", "1년에 차 비용 얼마?", "기름값 포함해서", "유지비 계산"
- **산출물:** 항목별(세금/보험/연료/정비/주차) 연간 유지비 추정
- **최소 데이터:** 차종, 연간 주행거리(없으면 15,000km 기본값)
- **깊이:** Flash + Deep-Dive 요청 시

### 7. Used Car Check (중고차 구매)
- **표현:** "중고차 살 때 뭐 봐야 해?", "중고차 체크리스트", "사기 안 당하려면", "성능기록부"
- **산출물:** 중고차 구매 단계별 체크리스트 + 사기 유형 주의
- **최소 데이터:** 없음 (예산/차종 있으면 맞춤형)
- **깊이:** Flash + Deep-Dive 요청 시

### 8. Insurance Compare (보험사 비교)
- **표현:** "보험사 어디가 좋아?", "다이렉트 vs 설계사", "보험 비교해줘", "갱신 vs 신규"
- **산출물:** 비교 포인트 + 비교 방법 (보험다모아 안내)
- **최소 데이터:** 없음
- **깊이:** Flash + Deep-Dive 요청 시

### 9. Traffic Fine (교통위반)
- **표현:** "신호위반 과태료?", "속도위반 벌점?", "주정차 딱지", "음주운전 처벌"
- **산출물:** 위반별 과태료·벌점·면허정지 기준 조견표
- **최소 데이터:** 위반 유형 (없으면 주요 위반 전체 제공)
- **깊이:** Flash only

### 10. Beginner Guide (입문)
- **표현:** "자동차보험 처음인데", "보험 종류 알려줘", "보험 개념이 뭐야"
- **산출물:** 자동차보험 구조 입문 Flash (초보자 모드 강제)
- **최소 데이터:** 없음
- **깊이:** Flash only

## 공통 인텐트 (공통 베이스에서 상속)

### 🔰 Beginner Mode 자동 전환
키워드: "쉽게", "기초", "초보", "처음인데", "개념"
→ 용어 설명 우선, 결론 먼저, 비유 활용

### ❓ Scope Out
자동차·보험 외 질문 → 안내 + 관련 스킬 추천
예: "운전면허 취득" → 도로교통공단 안내
예: "교통사고 소송" → law-search / legal-counsel 스킬 안내
