# 출처 신뢰도 티어

> 공통 Tier 구조 + 인용 형식: [`../../common/source_tiers_base.md`](../../common/source_tiers_base.md)

## Tier 1 — 최고 신뢰도 (공식)
- **행정안전부** / **위택스** (wetax.go.kr) — 자동차세 고지·납부 공식 기준
- **법제처** (law.go.kr) — 지방세법(자동차세), 도로교통법(교통위반)
- **금융감독원** — 자동차보험 표준약관, 보험 민원 기준
- **보험개발원** (kidi.or.kr) — 보험료 참고 공시 데이터
- **국토교통부** — 자동차 등록, 성능기록부 공식 시스템
- **경찰청** — 교통위반 과태료·벌점 공식 기준

## Tier 2 — 높은 신뢰도 (전문 매체)
- **보험다모아** (e-insmarket.or.kr) — 자동차보험 비교 공식 플랫폼
- **카히스토리** (carhistory.or.kr) — 중고차 사고 이력 공식 조회
- **엔카** / **KB차차차** — 중고차 시세 (시장가 참고)
- **손해보험협회** — 보험 통계·공시

## Tier 3 — 보강용
- 각 보험사 홈페이지 — 보험료 직접 견적 안내
- 자동차 커뮤니티 (보배드림, 클리앙) — 시세·경험 참고만

## 금지 출처
- 개인 블로그·카페 글 → 인용 금지
- 유튜브 자동차 채널 → 인용 금지 (참고만)
- 특정 보험사 광고성 자료 → 편향 가능, 출처 명시 필수

## 인용 형식
```markdown
# 법령
[지방세법](https://www.law.go.kr/법령/지방세법) 제127조 (시행 YYYY.MM.DD)

# 보험개발원 공시
(보험개발원 공시: 항목명, 기준일)

# 웹 출처
[엔카](https://www.encar.com) — 시세 참고
[보험다모아](https://www.e-insmarket.or.kr)

# 추정
"시세 추정" 명시 + 근거 병기
"보험료 추정 범위" + "실제는 조건에 따라 상이"
```
