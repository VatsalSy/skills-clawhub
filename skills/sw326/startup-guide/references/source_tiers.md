# 출처 신뢰도 티어

> 공통 Tier 구조 + 인용 형식: [`../../common/source_tiers_base.md`](../../common/source_tiers_base.md)

## Tier 1 — 최고 신뢰도 (공식)
- **법제처** (law.go.kr) — 업종별 허가·신고 법령
- **국세청 홈택스** (hometax.go.kr) — 사업자 등록 공식 절차
- **중소벤처기업부** (mss.go.kr) — 창업 정책·지원사업
- **창업진흥원** (kised.or.kr) / **K-스타트업** (k-startup.go.kr) — 창업 지원공고
- **소상공인시장진흥공단** (semas.or.kr) — 소상공인 지원
- **기업마당** (bizinfo.go.kr) — 정부 지원사업 통합 공시

## Tier 2 — 높은 신뢰도
- **대한상공회의소** — 창업 가이드
- **중소기업중앙회** — 소상공인 정책 정보

## Tier 3 — 보강용
- 스타트업 미디어 (플래텀, 벤처스퀘어) — 트렌드·공고 참고

## 금지 출처
- 창업 대행 광고성 블로그 → 인용 금지
- 유튜브 창업 채널 → 인용 금지

## 인용 형식
