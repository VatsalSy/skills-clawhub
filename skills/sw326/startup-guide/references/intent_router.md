## 인텐트 목록

### 1. Startup Checklist (창업 절차)
- **표현:** "창업하려면 뭐부터 해?", "창업 준비 절차", "창업 체크리스트"
- **산출물:** 아이디어 검증 → 사업자 등록 → 허가 → 운영 단계별 체크리스트
- **최소 데이터:** 없음 (업종 있으면 맞춤형)
- **깊이:** Flash + Deep-Dive 요청 시

### 2. Business Registration (사업자 등록)
- **표현:** "사업자 등록 어떻게?", "개인사업자 내는 방법", "홈택스 사업자 신청"
- **산출물:** 홈택스/세무서 등록 절차 + 필요 서류 + 소요 기간
- **최소 데이터:** 사업자 유형 (없으면 개인사업자 기준 안내)
- **깊이:** Flash only

### 3. License Check (허가·신고 조회)
- **표현:** "이 업종 허가 필요해?", "음식점 인허가", "영업 신고 어떻게?", "건강기능식품 판매"
- **산출물:** 업종별 허가·신고·등록 요건 + 담당 기관 + 법령
- **최소 데이터:** 업종명 (구체적일수록 정확)
- **깊이:** Flash + Deep-Dive 요청 시
- **처리:** law-search 스킬로 관련 법령 조회

### 4. Support Programs (정책자금·지원사업)
- **표현:** "창업지원금 뭐 있어?", "K-스타트업 공고", "소상공인 지원금", "TIPS 신청"
- **산출물:** 현재 모집 중인 지원사업 목록 + 신청 조건
- **최소 데이터:** 없음 (업종/단계 있으면 맞춤형)
- **깊이:** Flash + Deep-Dive 요청 시
- **처리:** korean-gov-programs 스킬 + web_search 보강

### 5. Business Verify (사업자 확인)
- **표현:** "이 사업자 살아있어?", "폐업했어?", "사업자 번호 확인", "거래처 사업자"
- **산출물:** company-info 스킬 호출 → 사업자 상태·업태·소재지
- **최소 데이터:** 사업자 번호 10자리
- **처리:** company-info 스킬로 위임

### 6. Tax Setup (창업 세금)
- **표현:** "창업하면 세금", "간이과세 vs 일반과세", "사업자 부가세", "원천징수"
- **산출물:** tax-guide 스킬 VAT Guide / Withholding 인텐트로 위임
- **처리:** tax-guide 스킬로 위임

### 7. Sole vs Corp (개인 vs 법인)
- **표현:** "개인사업자 vs 법인", "법인 설립해야 해?", "법인 전환 언제?", "절세 목적 법인"
- **산출물:** 세금·책임·비용·이미지·절차 5축 비교
- **최소 데이터:** 없음 (연 매출 있으면 절세 분기점 계산)
- **깊이:** Flash + Deep-Dive 요청 시

### 8. Startup Cost (창업 비용)
- **표현:** "창업 비용 얼마야?", "초기 자금 어느 정도?", "자본금 얼마?", "카페 창업 비용"
- **산출물:** 업종별 초기 비용 항목 추정 Flash
- **최소 데이터:** 업종
- **깊이:** Flash + Deep-Dive 요청 시

### 9. Beginner Guide (입문)
- **표현:** "창업 처음인데", "사업자가 뭐야?", "창업 종류 알려줘"
- **산출물:** 창업 유형·사업자 등록 개념 입문 Flash (초보자 모드 강제)
- **최소 데이터:** 없음
- **깊이:** Flash only
