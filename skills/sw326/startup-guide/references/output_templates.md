## Startup Checklist Flash

```markdown
### 🚀 창업 체크리스트 [업종명]

**Phase 1 — 사전 준비**
- [ ] 사업 아이디어 구체화 (타깃 고객, 수익 모델)
- [ ] 시장 조사 (경쟁사, 수요 확인)
- [ ] 자금 계획 (초기 자본금, 운영자금 최소 3개월치)
- [ ] 업종 허가·신고 요건 확인 → License Check 인텐트

**Phase 2 — 법적 설립**
- [ ] 개인사업자 vs 법인 결정 → Sole vs Corp 인텐트
- [ ] 사업장 확보 (임차 or 재택)
- [ ] 사업자 등록 (홈택스, 1~3일 소요)
- [ ] 업종별 허가·신고 완료
- [ ] 사업용 계좌 개설 (사업자 계좌)

**Phase 3 — 운영 준비**
- [ ] 세금 신고 방식 결정 (간이과세 vs 일반과세)
- [ ] 4대보험 가입 (직원 채용 시)
- [ ] 사업자 카드 발급 (비용 처리용)
- [ ] 회계 시스템 구축 (장부 기록)

**정책자금·지원금 검색** → Support Programs 인텐트
```

## Sole vs Corp Flash

```markdown
### 🚀 개인사업자 vs 법인 비교

| 항목 | 개인사업자 | 법인 (주식회사) |
|------|---------|--------------|
| 설립 기간 | 1~3일 (홈택스) | 2~4주 (법원 등기) |
| 설립 비용 | 거의 없음 | 최소 50~100만원 |
| 최소 자본금 | 없음 | 없음 (100원↑ 가능) |
| 세율 | 종합소득세 6~45% | 법인세 9~24% + 배당소득세 |
| 책임 범위 | **무한 책임** (개인 재산) | **유한 책임** (출자금 한도) |
| 신용도 | 낮음 | 높음 (투자·대출 유리) |
| 폐업 | 간단 | 청산 절차 필요 |

**법인 전환 검토 기준 (세금 분기점)**
- 연 순이익 약 **5,000만원↑** → 법인세율이 유리해지기 시작
- 연 순이익 약 **1억원↑** → 법인 전환 적극 검토 권장

⚠️ 절세 효과는 매출·비용 구조에 따라 다름. 세무사 상담 권장.
```

## Support Programs Flash

```markdown
### 🚀 창업 지원사업 현황 [단계: 예비/초기/성장]

**중앙부처 지원 (상시)**
| 기관 | 사업명 | 지원 내용 | 대상 |
|------|--------|---------|------|
| 창업진흥원 | 예비창업패키지 | 최대 1억원 사업화 지원 | 예비창업자 |
| 창업진흥원 | 초기창업패키지 | 최대 1억원 | 3년 이내 창업자 |
| 소상공인진흥공단 | 소공인 특화자금 | 저금리 대출 | 소상공인 |
| 기술보증기금 | 창업기업 보증 | 보증 지원 | 창업 기업 |

**공고 조회 방법**
- [K-스타트업](https://www.k-startup.go.kr) → 지원사업 공고
- [기업마당](https://www.bizinfo.go.kr) → 지원사업 검색
- [중소기업지원사업통합관리시스템](https://www.smes.go.kr)

⚠️ 지원사업은 공고 기간이 있습니다. 최신 공고 직접 확인 권장.
```
