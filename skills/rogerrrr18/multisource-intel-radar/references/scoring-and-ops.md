# Scoring & Ops Playbook

## Scoring rubric (0-100)
- Relevance (0-40): keyword + product-fit
- Actionability (0-30): can execute within 7 days
- Novelty (0-20): non-consensus / new mechanism
- Evidence (0-10): data, case, or direct source

## Decision labels
- 80+: Must-read (today)
- 60-79: Watchlist
- <60: Drop

## Digest format (CN)
- 1行总览
- Top3（洞察 + 为什么 + 今天动作）
- Watchlist 5条
- 被过滤噪音摘要（3条以内）

## Founder reminder
收藏 != 决策。
每条内容只允许两种命运：
1) 进入本周行动
2) 丢弃
