# Sentiment Report Template

Use this structure when compiling the final report.

## Header
```
# [Topic] 舆情分析报告
> 数据来源：[platforms used]
> 采集日期：YYYY-MM-DD
> 分析产品：[product list]
```

## Sections

1. **数据概览** — platform breakdown, total notes/comments/tweets
2. **各产品热度对比** — table with likes, collects, mentions per product
3. **情感分析** — positive/negative/concern percentages per platform
4. **各产品用户口碑详析** — per product: positive themes, negative themes, key quotes
5. **用户核心痛点** — grouped by theme (pricing, quality, ecosystem, design, privacy)
6. **竞争格局总结** — ranking across dimensions (price, brand, UX, ecosystem, design)
7. **产品启示** — actionable recommendations based on findings

## Quote Selection Criteria
- Pick comments with highest like counts (social proof)
- Include geographic diversity (ip_location)
- Balance positive and negative
- Prioritize specific, detailed feedback over generic praise/complaints
