# Hiring Criteria (2026)

## 1. Golang Engineer (All Levels)

### Common Requirements:
- **Core Tech**: Go, MongoDB, Redis, Kafka, gRPC, Microservices.
- **Experience**: 2-5 years relevant experience.
- **Competencies**: Business development, code quality, troubleshooting, fast learner.

### Level Specifics:
- **Junior**: Focus on syntax, standard library, and basic concurrency.
- **Middle**: Focus on system design, optimization, and project lifecycle.
- **Senior**: Focus on architecture, performance tuning, and leading technical initiatives.

## 2. PHP Engineer

### Requirements:
- **Core Tech**: PHP, MySQL, Redis, Frameworks (Laravel/Symfony).
- **Competencies**: Large-scale system maintenance, performance optimization.

## Scoring Rubric (0-100)
- **90-100**: Perfect fit; exceeds requirements in multiple areas.
- **75-89**: Strong candidate; matches all core requirements.
- **60-74**: Potential fit; some gaps in non-core areas.
- **<60**: Not a fit; major core skill missing.
