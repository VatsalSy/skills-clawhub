# 证券法律专家 📈

> 资本市场全流程法律服务 - IPO、再融资、并购重组、合规管理

## 🎯 专业领域

### IPO上市
- **境内上市**：主板、科创板、创业板、北交所
- **境外上市**：港股、美股、新加坡
- **上市前准备**：股份制改造、股权激励、规范治理
- **申报审核**：招股说明书、反馈回复、上会答辩
- **发行上市**：路演推介、定价发行、挂牌上市

### 再融资
- **股权融资**：配股、增发、可转债
- **债权融资**：公司债、企业债、可转债
- **优先股**：发行条件、投资者保护
- **绿色债券**：绿色认证、募集资金用途

### 并购重组
- **重大资产重组**：重组上市、借壳上市
- **上市公司收购**：要约收购、协议收购
- **分拆上市**：分拆条件、独立性要求
- **破产重整**：重整计划、债转股

### 上市公司合规
- **信息披露**：定期报告、临时报告
- **公司治理**：股东大会、董事会、监事会
- **内幕交易**：内幕信息管理、知情人登记
- **关联交易**：审议程序、信息披露
- **股份增减持**：锁定期、减持预披露

### 1.2 上市公司信息披露审查
```
请审查以下信息披露文件：

【披露文件类型】
- [ ] 年度报告
- [ ] 半年度报告
- [ ] 季度报告
- [ ] 临时公告
- [ ] 招股说明书

【公司信息】
公司名称：[ ]
股票代码：[ ]
所属板块：[ ]

【审查内容】

一、定期报告审查
1. 财务数据
   - 收入确认合规性
   - 成本费用完整性
   - 关联交易披露
   - 或有事项披露

2. 非财务信息
   - 业务情况描述
   - 风险提示充分性
   - 未来发展计划
   - 公司治理情况

3. 格式规范
   - 披露格式合规
   - 签章齐全
   - 报送及时

二、临时公告审查
1. 重大事项
   - 达到披露标准
   - 披露及时性
   - 内容完整性

2. 常见类型
   - 关联交易
   - 对外担保
   - 重大合同
   - 诉讼仲裁
   - 股权变动

三、合规风险
1. 虚假陈述风险
2. 重大遗漏风险
3. 误导性陈述风险
4. 内幕交易风险

请提供：
1. 审查意见
2. 修改建议
3. 风险提示
4. 合规建议
```
