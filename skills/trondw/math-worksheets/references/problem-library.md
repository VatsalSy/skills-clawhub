# Problem Type Library

Use this as a menu when designing problem sets. Pick types that target the student's current unit.

## Elementary / Pre-Algebra (Grades 5–8)

| Type | Example |
|---|---|
| Integer operations | $-8 + (-3)$, $(-4) \times 7$ |
| Order of operations | $3 + 2 \times (8 - 5)^2 \div 6$ |
| Fraction operations | $\frac{3}{4} + \frac{5}{6}$, $2\frac{1}{3} \times 1\frac{1}{2}$ |
| Decimal ↔ fraction ↔ percent | Convert 0.375 to fraction and percent |
| Ratios & unit rate | "60 miles in 1.5 hours — find mph" |
| Proportions | $\frac{x}{15} = \frac{4}{5}$ |
| Percent: find part/whole/rate | "35% of 80 is?" |
| One-step equations | $x + 7 = -3$, $\frac{x}{4} = 12$ |
| Two-step equations | $3x - 5 = 16$ |
| Inequalities | $2x + 3 > 11$, graph on number line |
| Coordinate plane plotting | Plot and label points in all 4 quadrants |
| Perimeter & area | Rectangles, triangles, composite shapes |
| Circle: circumference & area | Given radius or diameter |
| Volume | Rectangular prism, cylinder |
| Mean / median / mode / range | From a data set |
| Writing expressions from words | "5 less than twice a number" |
| Function tables | Complete $y = 2x + 1$ for $x = -2, 0, 1, 3$ |

## Algebra 1 (Grades 8–9)

| Type | Example |
|---|---|
| Slope from two points | Find slope through $(-1, 3)$ and $(4, -2)$ |
| Slope-intercept form | Write equation, graph the line |
| Standard form ↔ slope-intercept | Convert $3x - 2y = 12$ |
| Systems by substitution | Two equations, two unknowns |
| Systems by elimination | Same |
| Graphing systems | Find intersection point |
| Factoring GCF | $12x^3 - 8x^2 + 4x$ |
| Factoring trinomials ($a=1$) | $x^2 - 7x + 12$ |
| Factoring difference of squares | $x^2 - 49$ |
| Solving quadratics by factoring | $(x-3)(x+2)=0$ |
| Quadratic formula | With discriminant analysis |
| Graphing parabolas | Vertex, intercepts, axis of symmetry |
| Exponent rules | Product, quotient, power rules |
| Simplifying radicals | $\sqrt{72}$, $\sqrt{200}$ |

| Type | Example |
|---|---|

| Log properties | Product, quotient, power rules |
| Sequences | Arithmetic and geometric; find $a_n$ |
| Systems of 3 equations | Elimination method |
| Conic sections | Circles, parabolas — standard form |
