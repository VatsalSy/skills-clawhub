# Report Templates

## Executive Summary Template
- Performance headline: one sentence.
- 3 key metrics with context (current vs prior period).
- 2 notable drivers (positive/negative).
- 1 clear next action.

## KPI Dashboard Template
Include cards for:
- Total Revenue
- Growth Rate
- Total Orders
- Average Order Value
- Top Segment/Product

## Detailed Analysis Prompts
- Which dimensions drive most variance?
- Are trends stable, seasonal, or volatile?
- Which categories underperform despite volume?
- Any concentration risk (few customers/products dominating)?

## Recommendations Template
1. Keep: what is working and should continue.
2. Fix: what is underperforming and likely controllable.
3. Test: one experiment with measurable success criteria.
4. Watch: one risk metric for early warning.

## Example User Requests
- Generate a sales report from this CSV.
- Create a monthly KPI dashboard for leadership.
- Build an executive summary with trend charts.
- Produce a data analysis report and recommendations.
